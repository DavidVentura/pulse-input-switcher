# pulse-input-switcher

Example output

```
Programs
index     sink      iconname  appname   
==================================================
1         0         Telegram  Telegram  
2970      0         mumble    Mumble    

Sinks
index                  desc                   
=====================================================================
0                      Analog Stereo          
5                      Digital Stereo (HDMI)  

```


running ```./audio.py 2970 5 ``` would swap Mumble to the HDMI output.
